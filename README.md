# roommate-app
# Start Commands
Node server.js

npm run serve

Start/Stop server:
service postgresql start/stop

Server Status:
service postgresql status
# Postgresql Commands
PSQL Commands:
sudo su postgres
psql

list all database: \lu

list all users & roles: \du

database username: userDatabase
database password: 123qwer456

Login:
sudo -u - postgres psql
